# Front-end (Next.js)

## Funcionalidades
- Login e cadastro (JWT)
- Listar tarefas
- Criar tarefa
- Alterar status
- Excluir tarefa
- Filtro por status
- Loading e tratamento de erro
- Layout responsivo básico

## Rodar local
```bash
cd parte2/front-end
npm install
cp .env.example .env.local
npm run dev
```

A aplicação abre em `http://localhost:3000`.

## Testes unitarios
```bash
cd parte2/front-end
npm install
npm run test
```
